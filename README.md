To-Do List App
A simple console-based To-Do List application developed in C++.
Features
- View Tasks
- Add Tasks
- Delete All Tasks
- User-friendly menu interface

Technologies Used
- C++
- Standard Library (iostream, string)

How to Run
1. Compile the source file:
   g++ To_Do_List.cpp -o todo

2. Run the program:
   ./todo
